# CollateralIQ Pro

## Overview

CollateralIQ Pro is an AI-powered collateral intelligence platform developed during TenzorX 2026 to support secured lending and collateral assessment workflows.

The platform combines:

* Geospatial verification
* Property valuation intelligence
* Distress value estimation
* Loan-to-Value (LTV) analysis
* Explainable AI insights
* AI-generated underwriting summaries

to help financial institutions make faster and more informed lending decisions.

## Installation

Clone the repository:

git clone <repository-link>

Navigate to project folder:

cd CollateralIQPro

Create virtual environment:

python -m venv venv

Activate virtual environment:

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py

---

## Required API Keys

The following API keys are required:

1. OpenCage API Key
2. Google Gemini API Key

Store keys inside environment variables or configuration file before execution.
